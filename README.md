# SentinelChain AI: Autonomous Supply Chain Risk Intelligence Platform

SentinelChain AI acts as a cutting-edge **autonomous supply-chain intelligence control plane**. Engineered for logistics officers, the platform continuously monitors global threats (seaports lockouts, climate anomaly vectors, geopolitical tensions) in real-time, models impact scopes through a unified operations Digital Twin, dynamically recalculates predictive risk risk metrics, and suggests optimal alternative supplier candidates using serverless AWS-native architecture schemas.

---

## 🚀 Key Architectural Features Included

1. **Executive Dashboard Cockpit**
   - Live metrics summary panels displaying Cumulative Financial Exposures, Severe Alerts count, Ocean Leg delays, and Average Supplier ratings.
   - Comprehensive interactive line charts plotting Predictable Exposure and active Threats trends over time using **Recharts**.

2. **Global Risk Center**
   - Beautiful custom-rendered **vector world map (SVG)** plotting exact coordinates of suppliers and active hazards (climate beacons, strike locks).
   - Real-time interactive point-pin filtering categories and instant details sheets with voice brief hooks.

3. **Autonomous Sourcing Recommender**
   - Compares current threatened primary suppliers against domestic alternative candidates.
   - Dynamic comparative scorecard tracking Delivery Lead times, Cost index factors, Reliability averages, and ESG sustainability indexes.
   - **Execute Swap Trigger**: Interactive state mechanism that replaces nodes dynamically and escalates target alerts to "Mitigating".

4. **Digital Twin Network Topology**
   - Map-node diagram visualizing connected chains starting from raw fabrication in Asia to logistics hubs and gigafactory grids in North America/Europe.
   - Interactive capacity controllers and safety hand-on safety slider parameters.

5. **Risk Assessment Recalculations Engine**
   - Test risk stress levels by dragging parameter sliders (geopolitical risks, extreme storms, wildcat labor rates) and observe live recalculations of cumulative exposures and shipping delay curves.

6. **Voice AI Briefing Terminal**
   - Future-proof emulator replicating **Amazon Nova Sonic**. Try asking custom questions or clicking preset log cards to hear real-time spoken transcripts backed by glowing wave-animations.

7. **AWS Architecture Blueprint**
   - Detailed, interactive diagnostic map describing step-by-step how **Amazon S3, Amazon Nova Embeddings, Bedrock Agents, EventBridge, Lambda, and SNS** communicate seamlessly.
   - Includes real JSON REST payloads schemas useful for system judges.

## 🛠️ Technology Stack & Libraries

- **Frontend Core**: React (Functional Components, Custom Context hooks)
- **Compilation Engine**: Vite, TypeScript
- **Styling Architecture**: Tailwind CSS
- **Motion Orchestrations**: Framer Motion (imported from `motion/react`)
- **Analytical Charts**: Recharts
- **Icon Libraries**: Lucide Icons
- **Routing Engine**: React-Router-Dom (`HashRouter` for sandboxed iframe-safeties)

## 💻 Technical Execution & Run Guides

The application compiles with **zero syntax errors** or warning blocks. Run immediately using standard operations directives:

### 1. Installation
Install all required packages:
```bash
npm install
```

### 2. Development Execution
Launch the high-performance local dev server on port `3000`:
```bash
npm run dev
```

### 3. Production Compilation
Build compiled static optimized files in the `/dist` directory:
```bash
npm run build
```
